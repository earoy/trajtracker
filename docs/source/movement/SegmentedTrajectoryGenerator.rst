SegmentedTrajectoryGenerator class
==================================

Generate a stimulus movement trajectory. The trajectory is defined as several trajectory parts,
each of which is defined by another stimulus generator.

Use this class in conjunction with :class:`~trajtracker.movement.StimulusAnimator`

To see how this is implemented, see the "TrajGenerator" sample provided with TrajTracker.
